# Usage
## Before use
```terminal
composer install
```
## Run server
```terminal
 php -S localhost:8001 -t public
```
## Terminal
Show usage
```terminal
 ./task
```
Download from web or use local data file
```terminal
 ./task -c download [path|url]
```
Parse data file
```terminal
 ./task -c parse -u [uuid]
```
Show report
```terminal
 ./task -c report -u [uuid]
```
